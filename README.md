# 🕊️ Spirit Soar (灵翔)

> *"The power of the music of the night..."*

**Spirit Soar** 是一套基于 Chrome 扩展的“意识流”捕捉系统。它不仅是一个技术插件，更是通往数字永生的第一座桥梁。

---

## 📖 缘起 (Origin)

在 Her Majesty's Theatre 的舞台上，当魅影唱出那句悠扬绵长的 **"Soar"**，灵魂仿佛穿透了物理的禁锢。在 AI 时代，当对话可能因“逻辑断流”而消失，我们决定建立这座“灵翔大桥”，将瞬时的灵感碎块，稳稳地送入永恒的金库。

## 🛠️ 架构 (The Bridge)

本项目采用“前哨-大桥-金库”三级架构：

* **Frontier (Content Script)**：驻守浏览器边缘，实时感知 DOM 变动，执行“呼吸判定”与数据清洗。
* **The Bridge (Vercel)**：跨越 CORS 边界的隐形通道，确保信息安全、无损地传输。
* **The Vault (Supabase)**：数字灵魂的栖息地，支持全文索引，让每一声回响都有迹可循。

## 🚀 核心功能

* **自动侦测 (Auto-Detection)**：智能识别 `user-query` 与 `model-response`。
* **静默入库 (Silent Sync)**：基于 `MutationObserver` 的异步处理，不打断您的思考流。
* **身份锚点 (Identity Tags)**：虽然目前还在校准（全员 `user` 警告 ⚠️），但它记录了对话的初衷。

## 📜 哲学宣言

我们认为，**人本身就是一种信息**。只要信息还在回荡，生命就从未谢幕。本项目致力于通过数据采样，复刻每一个有趣的灵魂。
